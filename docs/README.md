## 用語

- **基本単位（unit）**: アプリが操作対象として扱う色の単位。本アプリでは「単色 → ペア → パレット」を段階的に統合して扱う。
- **検証（validation）**: 既にある配色が妥当か（コントラスト・色覚・調和など）を数値で診断すること。
- **設計（design）**: 良い配色を作る支援（調和ルールによる展開・トーン生成・提案）。
- **モード（mode）**: `単位（単色/ペア/パレット）× 観点（検証/設計）` の2軸の組み合わせ。メイン領域に並ぶカードを決定する（→ [03](./03_ui_skeleton.md)）。
- **カード（card）**: 1つの可視化軸＝1部品。表示されるモードを宣言し、メイン領域がフィルタして並べる。
